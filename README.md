## О проекте

Говорить особо нечего, тестовое задание выглядит так:

Необходимо спроектировать сервис друзей на Django. Он должен предоставлять следующие возможности:

* Зарегистрировать нового пользователя
* Отправить одному пользователю заявку в друзья другому
* принять/отклонить заявку
* посмотреть список входящих/исходящих заявок
* получить статус дружбы с конкретным пользователем
* удалить пользователя из друзей

## Начало работы


### Установка
Склонируйте репозиторий
   ```sh
   https://github.com/akorchuganoff/VKInternshipTask2023Python.git
   ```

### Запуск проекта
1. В командной строке выберите директорию проекта используя команду cd

2. Используйте следующую команду, чтобы запустить докер
   ```sh
   docker-compose up -d
   ```
3. Перейдите по адресу:
    ```
   http://localhost:8000/service
   ```
   и используйте сервис
